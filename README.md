# 🚀 Presentation

**KERNEL TOOL** is a modern Python cybersecurity multitool designed for fast access to multiple offensive and investigative utilities inside one terminal interface.

The project combines:
- OSINT tools
- Pentest utilities
- DNS scanners
- Username checkers
- Email investigation systems
- Network analysis
- DDoS testing utilities
- IP utilities
- Recon systems

The interface is designed with:
- Animated terminal UI
- Cybersecurity style design
- Rich live rendering
- Fast navigation system
- Modular architecture

The goal of the project is to create an all-in-one cyber framework for researchers, developers and advanced users.

# 📦 Installation

# 🐧 Linux Installation

## 1 — Install Git

### Debian / Ubuntu
```bash
sudo apt update && sudo apt install git -y
```

### Arch Linux
```bash
sudo pacman -S git
```

### Fedora
```bash
sudo dnf install git
```

---

## 2 — Clone the Repository

```bash
git clone https://github.com/LXKCV/KERNEL-TOOL/
```

---

## 3 — Open the Folder

```bash
cd KERNEL-TOOL
```

---

## 4 — Install Python Dependencies

```bash
pip install -r requirements.txt
```

---

## 5 — Launch the Tool

```bash
python home.py
```
