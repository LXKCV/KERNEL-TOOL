# KERNEL TOOL

A terminal multitool built for Linux. (and Windows)

KERNEL TOOL is a modular interface designed to provide system tools, network utilities, and advanced terminal interactions inside a visually immersive console environment.

This project focuses on creating a **control interface** rather than a simple command-line tool.

## Current Features

- Animated terminal interface
- Cyberpunk ASCII title
- Blue glow / shimmer animation
- Keyboard navigation system
- Modular architecture
- Interactive menu system
- Terminal-based UI

---

## Planned Features

The following modules will be implemented in future updates:

- System Monitor (CPU / RAM / Disk usage)
- Network Scanner
- Port Scanner
- Password Generator
- ASCII Text Generator
- Integrated Command Console
- Logs Viewer
- Settings Panel
- Theme Customization
- Animation Controls
- Process visualizer
- Network activity monitor
- Plugin module loader

More advanced modules will be added over time as the project evolves.
