# KERNEL TOOL

A terminal multitool built for Linux. (and Windows)

KERNEL TOOL is a modular interface designed to provide system tools, network utilities, and advanced terminal interactions inside a visually immersive console environment.

This project focuses on creating a **control interface** rather than a simple command-line tool.
